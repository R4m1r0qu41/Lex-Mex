# Artículo 257

Cuando el Comandante local no cuente con el suficiente personal integrante del cuerpo a que se refiere el artículo [244](articulo-244.md) del presente Reglamento para desplegar operativos de seguridad, así como para atender [Situaciones de Emergencia](articulo-10.md), podrá disponer para esos efectos del personal administrativo calificado de la [Capitanía de Puerto](articulo-10.md).
