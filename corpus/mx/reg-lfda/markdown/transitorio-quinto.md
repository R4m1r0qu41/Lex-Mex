# QUINTO

El [Registro](articulo-2o.md) conservará todos los expedientes y documentos de las [Sociedades](articulo-2o.md) de Autores que hayan sido registrados con base a la [Ley](articulo-2o.md) anterior, pero no podrá registrar ningún documento nuevo de estas sociedades y se reservará únicamente para las sociedades de gestión colectiva que se constituyan conforme a la Ley vigente.
