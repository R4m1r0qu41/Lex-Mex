# Artículo 98

La calificación de créditos cuyo saldo se encuentre parcial o totalmente cubierto en (111) 144

relación con el valor de las garantías obtenido conforme al Artículo [96](articulo-96.md), fracción I podrá arrojar como resultado distintas calificaciones, una para la parte expuesta del saldo del crédito y una o más para la porción o porciones cubiertas, dependiendo del número de garantías utilizadas y las características de las mismas.
