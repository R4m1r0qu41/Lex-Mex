# SÉPTIMA

La Comisión Nacional Bancaria y de Valores y el Banco de México contarán con un plazo de doce meses para la emisión conjunta de las disposiciones a que se refieren los artículos 48, 54, 56, segundo párrafo de esta Ley, así como para celebrar el convenio previsto en el artículo 71 de este ordenamiento.
