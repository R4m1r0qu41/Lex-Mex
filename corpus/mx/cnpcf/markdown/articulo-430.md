# Artículo 430

Para el examen de los testigos, se observarán las formalidades que para esta prueba regula el presente [Código Nacional](articulo-2.md).

La autoridad jurisdiccional podrá ampliar el examen de los testigos con las preguntas que estime pertinentes.
