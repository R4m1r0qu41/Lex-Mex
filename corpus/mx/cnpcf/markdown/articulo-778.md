# Artículo 778

Si hubiera oposición contra el proyecto, se substanciará en forma incidental, procurando que, si fueren varias, la audiencia sea común, y a ella concurrirán las personas interesadas y el partidor para que se discutan las gestiones promovidas y se reciban pruebas.

Para dar curso a esta oposición, es indispensable expresar concretamente cuál sea el motivo de la inconformidad y cuáles las pruebas que se invocan como base de la oposición.
