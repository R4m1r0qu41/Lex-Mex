# NOVENO

Auxilio procesal Cuando una autoridad penal reciba por exhorto, mandamiento o comisión, una solicitud para la realización de un acto procesal, deberá seguir los procedimientos legales vigentes para la autoridad que remite la solicitud, salvo excepción justificada.
