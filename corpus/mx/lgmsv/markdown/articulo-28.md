> Título SEGUNDO · Capítulo IV

# Artículo 28

Integración de Indicadores y Bases de Datos de [Movilidad](articulo-3.md) y Seguridad Vial. La integración de indicadores y bases de datos del [Sistema de Información Territorial y Urbano](articulo-3.md) se integrarán por las siguientes:

I. Base de Datos sobre información de movilidad, y

II. Base de Datos de información y seguimiento de seguridad vial.
